LE LOGICIEL TOR:

Tor est un réseau informatique superposé mondial et décentralisé. Il se compose d'un certain nombre de serveurs, appelés nœuds du réseau et dont la liste est publique. Ce réseau permet d'anonymiser l'origine de connexions TCP. Tor est un outil fiable pour naviguer sur le Web. Mais vous devez faire attention. Les scripts sur Tor peuvent être utilisés pour exposer vos informations. Les FAI peuvent voir que vous utilisez Tor sans VPN.Le principe de Tor est à peu près le même que les VPN. Là aussi votre trafic passe par le réseau Tor avant d'arriver aux services internet. La différence réside dans le fait que vous ne passez pas par un seul serveur. On passe par les relais suivants : Le garde à l'entrée, le relai du milieu et enfin le nœud de sortie.

QU'EST-CE QU'ON TROUVE SUR LE DARK WEB?

On dit souvent que sur le dark web on trouve des contenus dangereux et choquants, c'est en partie vrai mais le dark web abrite toutes sortes d'informations qui ne sont dark, qu'en raison de leur difficulté d'accès.
